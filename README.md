# Stitch 3D Keychain

## Project Overview

This project is a 3D printable Stitch keychain designed using Onshape.
The model was created as a lightweight decorative keychain suitable for 3D printing.

---

## Features

- 3D printable STL model
- Designed using Onshape
- Lightweight and compact design
- Smooth edges for easy printing
- Decorative keychain

---

## Software Used

- Onshape
